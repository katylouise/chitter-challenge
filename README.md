# Chitter

A Twitter-style app built using Ruby, Sinatra and DataMapper ORM. Tested with RSpec and Capybara.

## Getting Started

```
git clone https://github.com/katylouise/chitter-challenge.git
cd chitter-challenge
bundle install
rackup
```
Go to localhost:9292

## Running Tests

To run rspec:
```
rspec
```
